## Comparison Based Sorts
1) Bubble Sort          - O(n2)
2) Insertion Sort       - O(n2)
3) Selection Sort       - O(n2)

4) Quick Sort                                 - O(nlogn)
5) Merge Sort (Iterative & recursive )        - O(nlogn)
6) Shell Sort                                 - O(nlogn)

## Index Based Sorts
7) Bin/Bucket Sort      - O(n)
8) Radix Sort           - O(n)
